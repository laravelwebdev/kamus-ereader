# Kamus E-Reader

1. Kamus besar bahasa indonesia versi 6 update oktober 2025
2. Indonesia-Inggris (63.272 Kata)
3. Inggris-Indonesia (115.780 kata + synonym + Irregullar Verb)

# Terms
1. Tidak untuk penggunan secara komersial
2. Tujuan saya buat murni hanya untuk memberikan pengetahuan berbahasa Indonesia khususnya bagi pengguna ereader
3. Semoga nantinya tersedia KBBI resmi untuk ereader.

# Supported Format
1. xdxf (tested)
2. stardict (tested on Linguae, Stradict, Meebook)
3. dic (tested on Pocketbook Era)
4. mobi (Untuk Kindle, not tested yet, gak punya kindle)
5. kobo (not tested yet, gak punya kobo)
